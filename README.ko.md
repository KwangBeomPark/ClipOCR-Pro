# 🚀 다운로드 및 실행 방법 (Quick Start)

**ClipOCR-Pro**는 별도의 귀찮은 설치 과정 없이, 다운로드 후 더블 클릭만으로 즉시 동작하는 **무설치 포터블(Portable) 프로그램**입니다.  
프로그래밍 지식이 없거나 컴퓨터 설정이 서툰 일반 팀원들도 복잡한 설치 없이 원클릭으로 바로 업무에 활용할 수 있도록 배포에 초점을 맞추어 설계되었습니다.

### 📥 일반 팀원용 (원클릭 무설치 다운로드)
1. 깃허브 화면 우측의 **[Releases]** 탭을 클릭하여 이동합니다.
2. 가장 최신 릴리즈에 등록된 **`ClipOCR-Pro.zip`** 또는 **`ClipOCR-Pro.exe`** 단독 실행 파일을 다운로드합니다.
3. 압축을 풀고 **`ClipOCR-Pro.exe`** 파일을 더블 클릭하여 실행하면 윈도우 하단 트레이 영역에 아이콘이 활성화되며 즉시 동작합니다!
   - *💡 **팀 생산성 확장 팁**: 공통적으로 사용하는 환경 설정(`config.ini`)을 한 명의 관리자가 세팅한 뒤, 해당 설정 파일을 동료들에게 배포하여 동일한 폴더에 넣고 실행하게 하면 즉시 **부서 전체의 업무 방식 상향 표준화**가 가능합니다.*

### 🛠️ 파워 유저 및 개발자용 (커스텀 빌드)
본 소프트웨어는 투명하게 소스코드가 전면 공개된 **100% 오픈소스**입니다. 직접 기능을 추가하거나 고도화하고 싶다면 아래 방식을 참고하세요.
1. [AutoHotkey v2](https://www.autohotkey.com/)를 설치합니다.
2. 이 저장소를 Clone하고 `ClipOCR-Pro.ahk` 소스 코드를 필요에 맞게 커스텀합니다.
3. 코드를 수정한 뒤 AutoHotkey 컴파일러(Ahk2Exe) 등을 통해 본인만의 아이콘과 메타데이터가 적용된 커스텀 `ClipOCR-Pro.exe`를 직접 패키징할 수 있습니다.

# 💼 실제 업무 활용 예시
- **정산 및 증빙 검토**: 핵심 내용을 표시하여 승인자가 빠르게 확인
- **해외 메일 및 증빙 번역**: 선택 텍스트 번역 또는 Google 이미지 번역 활용
- **다중 자료 비교**: ERP / Excel / 메일 / 증빙을 동시에 띄워 비교
- **리포트 작성**: 여러 참고 자료를 화면에 고정하여 활용
- **회의 및 교육**: 매뉴얼 일부를 캡처 후 접기/펼치며 설명
- **이미지 번역**: 메일 첨부 이미지 및 스캔 문서를 Google 이미지 번역으로 확인
- **문서 첨부 최적화**: 캡처 이미지를 앱 설정값에 따라 크기 조정하여 붙여 넣기

# ⌨️ 주요 단축키

| Shortcut | 기능 |
|---------|------|
| `Win + Drag` | 화면 캡처 후 최상단 플로팅 |
| `Win + CapsLock` | 선택한 텍스트를 Google 번역 API로 번역 |
| 플로팅 화면에서 `Mouse Right` | 이미지 번역, 주석, 복사/저장, 창 관리 메뉴 열기 |
| 플로팅 화면에서 `Double Click` | 최소화 / 복원 |
| 플로팅 화면에서 `Ctrl + C` | 플로팅 이미지 복사 |
| 플로팅 화면에서 `Shift + Drag`, `Ctrl + Drag`, `Alt + Drag`, `Ctrl + Z`  | RedBox, Yellow, Green, Undo |
| 플로팅 화면에서 `Ctrl + ↑(Up)`, `Ctrl + ↓(Down)`, `Ctrl + ←(Left)`, `Ctrl + Esc` | (플로팅 이미지 조정) 최소화, 원래크기, 왼쪽 정렬, 전체 닫기 |
