저는 **전문 개발자가 아니라 재무부서에서 근무하는 실무자**로
반복적인 **화면 캡처, OCR, 문서 정리, 정보 공유** 업무를 줄이기 위해 개발을 시작했고,
작은 자동화 스크립트는 점차 실무 중심 업무 도구로 발전했습니다.

> **반복 업무 감소 · 팀 생산성 향상 · 실무 자동화 · 문제 해결**
을 목표로 만든 업무 지원 도구입니다.


---

# 💼 실제 업무 활용 예시
- **정산 및 증빙 검토**: 핵심 내용을 표시하여 승인자가 빠르게 확인
- **해외 메일 및 증빙 번역**: OCR 또는 Google 이미지 번역 활용
- **다중 자료 비교**: ERP / Excel / 메일 / 증빙을 동시에 띄워 비교
- **리포트 작성**: 여러 참고 자료를 화면에 고정하여 활용
- **회의 및 교육**: 메뉴얼 일부를 캡처 후 접기/펼치며 설명
- **이미지 텍스트 추출**: 메일 첨부 이미지 및 스캔 문서 OCR
- **문서 첨부 최적화**: 캡처 이미지를 앱 세팅 값에 딸 크기 조정하여 붙여 넣기

# ⌨️ 주요 단축키

| Shortcut | 기능 |
|---------|------|
| `Win + Drag` | 화면 캡처 후 최상단 플로팅 |
| `Win + Ctrl + Drag`, `Win + Alt + Drag` | 한국어/영어 OCR, 폴란드어/영어 OCR |
| 플로팅 화면에서 `Mouse Right` | 구글 번역 이미지/텍스트 선택 |
| 플로팅 화면에서 `Double Click` | 최소화 / 복원 |
| 플로팅 화면에서 `Ctrl + C` | 플로팅 이미지 복사 |
| 플로팅 화면에서 `Shift + Drag`, `Ctrl + Drag`, `Alt + Drag`, `Ctrl + C`  | RedBox, Yellow, Green, Undo |
| 플로팅 화면에서 `Ctrl + ↑(Up)`, `Ctrl + ↓(Down)`, `Ctrl + ←(Left)`, `Ctrl + Esc` | (플로팅 이미지 조정) 최소화, 원래크기, 왼쪽 정렬, 전체 닫기 |
